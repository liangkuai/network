# HTTPS

### 为什么需要 HTTPS？

因为 HTTP 不安全，明文传输，并且整个传输过程完全透明，可以轻松截获、修改或者伪造请求/响应报文。导致 3 种风险：

- 窃听风险
- 篡改风险
- 伪装风险（主要是伪装成服务端）

而 HTTPS 具有 3 种特性来保证安全性：

- 加密（防窃听）
- 完整性保护（防篡改）
- 认证（防伪装）


### HTTPS 的实现基础

HTTPS 把下层的传输协议由 TCP 协议改成了 SSL/TLS 协议，由 SSL/TLS 协议实现安全通信。
